怎么安装第三方包？

有些同学可能不知道怎么安装第三方包，其实一般情况下Anaconda已经集成了几乎所有我们常用的第三方包，但是Pygal这个包却没有，所以这里以Pygal为例教大家怎么安装第三方包。

Win版操作

如果你只安装了Anaconda，打开Anaconda Prompt，输入pip install pygal

就这么简单！

如果你既安装了Anaconda，又安装了原版的Python，为了避免冲突，我们稍微修改一下Anaconda的安装文件D:\Anaconda\Scripts （这个是我的路径，你要找你的安装路径！）把pip.exe和 pip-script.exe改成condapip.exe和condapip-script.exe

然后要给原版Python安装pygal，请右键开始菜单，然后点击运行（run），然后输入cmd，在弹出的terminal里面输入pip install pygal。

如果给anaconda安装，就还是用Anaconda Prompt, 然后输入condapip install pygal

最后，如果你想查看一下Anaconda有哪些第三方包，就在Anaconda Prompt里面输入conda list

Mac版操作

Mac我只安装了Anaconda，所以打开terminal（请在你的launchpad里面找找），然后输入pip install pygal就欧了。
